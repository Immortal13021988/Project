# Проект к разделу 2

## Описание проекта:

1. Реализованы функции маскировки номера карты и номера счета к уроку 9.1.
2. Реализованы функция маскировки карт и счетов, а так же функция преобразования формата даты кк уроку 9.2
3. Реализована функция фильтрации словарей в списке по заданному аргументу и функция сортировки с писка словарей по дате
   к уроку 10.1
4. Реализованы функция фильтрации транзакций по заданному параметру, функция, выводящая описание транзакции, генератор
   номеров карт в соответствующем формате к уроку 11.1
5. Реализован декоратор логирования выполнения функции к уроку 11.2
6. Реализованы функция считывания файла JSON и функция конвертации транзакций в рубли к уроку 12.1
7. Реализованы функции считывания файлов CSV и XLSX к уроку 13.1
8. Реализованы функция фильтрации транзакций и функция подсчета кол-ва транзакций по заданному описанию к уроку 13.2
9. Реализована дополнительная функция вывода видов описания транзакций в файлах.

## Установка:

1. Клонируйте репозиторий:

git clone https://github.com/Immortal13021988/Project/tree/feature/homework_12_1

2. Установите зависимости:

pip install -r requirements.txt

## Использование:

1. Запустите файл main.py
